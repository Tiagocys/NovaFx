Como ser um parceiro NovaFx

Primeiro passo

Abertura de conta

Ao criar uma conta parceiro NovaFx na FBS teremos acesso aos seus dados como o seu nome completo e números das suas contas. Você pode escolher entre dois tipos de conta: Cent e Standard. A conta Cent é aconselhável para novos investidores que pretendem investir valores menores que $100, a Standard é para investidores com maior poder de investimento. O algotrading foi desenvolvido para os dois tipos de conta, escolha a melhor para você e comece com investimentos a partir de $1. Não esqueça de criar uma conta para a plataforma Metatrader 4.
